# Universal AI Data Analytics Studio

## Overview

Universal AI Data Analytics Studio is an interactive data analytics platform built with Python and Streamlit. It enables users to upload datasets, perform data cleaning, explore data through visualizations, and generate analytical insights from a user-friendly dashboard.

## Features

### Data Management

* Upload CSV and Excel datasets
* Preview dataset contents
* Display dataset information and statistics

### Data Cleaning

* Handle missing values
* Remove duplicate records
* Data preprocessing utilities

### Exploratory Data Analysis (EDA)

* Summary statistics
* Correlation analysis
* Distribution analysis
* Outlier detection

### Data Visualization

* Interactive charts and graphs
* Histograms
* Bar charts
* Pie charts
* Scatter plots
* Correlation heatmaps

### Analytics Dashboard

* Real-time data insights
* Interactive filtering
* Visual reporting

## Installation

1. Clone the repository

```bash
git clone https://github.com/anjaliaajagiya5277-creator/Universal_AI_Data_Analytics_Studio.git
```

2. Navigate to the project directory

```bash
cd Universal_AI_Data_Analytics_Studio
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
